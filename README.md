# Terraform GCP Infrastructure

This repo demonstrates Infrastructure as Code (IaC) on Google Cloud using Terraform.

## Features
- VPC & Firewall
- Compute Engine VMs
- GCS Buckets (with static website module)
- Remote state with GCS
- Module usage (instances, storage)

## Usage
terraform init
terraform plan
terraform apply


## Cleanup
terraform destroy
